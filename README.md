# ⚔️ Arogya Yodha — आरोग्य योद्धा

A story-based gamified health learning web app for Indian school children (Age 6–16).

**Phase 1 MVP** — Hygiene Pillar · "Arjun and the Germ Demon"

## 🚀 Deploy on Vercel (Free)

1. Push this folder to GitHub
2. Go to https://vercel.com → New Project → Import from GitHub
3. Framework: **Vite** (auto-detected)
4. Click Deploy — live in ~60 seconds!

## 💻 Run Locally

```bash
npm install
npm run dev
```

## 🏗 Tech Stack
- React 18
- Vite 5
- Tailwind CSS (CDN)

## 📦 Build
```bash
npm run build
```
